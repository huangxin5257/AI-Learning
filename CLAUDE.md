# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Purpose

This is a knowledge-base repository for AI/LLM learning materials, written primarily in Chinese. It is not a software project — there are no build steps, tests, or linting commands.

## Content

The repository contains reference documents on prompt engineering and LLM usage:

- **Prompt_Engineering_失效模式识别与治理_全览.md** — A comprehensive guide covering ~25 LLM prompt failure modes across 8 categories (instruction following, knowledge/facts, format/structure, semantic understanding, context management, reasoning/logic, role boundaries). Each failure mode includes detailed explanations, remediation strategies, and 3 worked examples (original prompt → failure output → improved prompt). Ends with a quick-reference summary table.

## Working with This Repository

- All content is in Markdown (Chinese language).
- When editing or adding content, maintain the existing document structure: section headers, bold labels for 详细解释/解决方法/示例, code blocks for example prompts, and `---` separators between examples.
- The primary audience is Chinese-speaking practitioners learning prompt engineering techniques.
