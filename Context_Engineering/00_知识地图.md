# Context Engineering 知识地图

## 这份知识库是什么

一套从零开始学习 Context Engineering（上下文工程）的中文教程。目标是让你从"会写 prompt"进化到"会设计 context 系统"。

## 贯穿案例：客服问答助手

整个学习路线用同一个任务串联——**为一家电商平台构建智能客服问答助手**。

你会看到同一个任务在不同阶段的做法差异：

- **M0 直觉入门**：随手写一句 prompt → 发现回答不稳定、幻觉频出 → 意识到需要"工程化"
- **M1 基础概念**：理解 token 限制、信息衰减、context 来源 → 知道为什么有些做法有效、有些无效
- **M2 核心操作与实战**：用 System prompt 分层架构 + RAG 检索注入 + 多轮管理 → 构建一个生产可用的方案

每一层的知识不是替代前一层，而是在前一层的基础上叠加。
